##Tutorials to create webpages with verification

a tutorial is a set of html documents in a folder in the tuto folder. it s also an element in tuto.json folder.index.html is the beginning
The element has key that are

* Name of tutorial (display in WebSpree)
* Folder path to open the folder
* Autors
* Ordered list of files to follow the tutorial
* in addition to the path where the files is stored there is
the verification
* (language(s))
* date
* level
* (bool)showlink to next
* more ?
