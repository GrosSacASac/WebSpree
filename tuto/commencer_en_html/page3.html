Adresse absolue et adresse relative

Une adresse absolue est la même et pointe la même chose de n'importe où on se trouve. Une adresse relative change en fonction du lieu de départ mais elle est plus brève. L'adresse relative est aussi constante si tout un site web change de serveurs. Une adresse relative indique le chemin depuis la page sur laquelle on se trouve. On utilise / pour rentrer dans un dossier et /.. pour rentrer dans le dossier parent. Ainsi <a href="/../salut.html" >Exemple relatif</a> va vous mener sur le fichier salut.html dans le dossier parent et <a href="/a/b/bye.html" >Exemple relatif 2</a> va vous mener au fichier bye.html qui se trouve dans le dossier b qui se trouve dans le dossier a. L'adresse relative d'un fichier dans le même dossier est simplement le nom du fichier. Les adresses absolue , requirent le protocole comme http:// par exemple

En pratique on utilise les adresses absolues pour les fichiers sur les lesquels on a le contrôle dans notre dossier du site web. Pour ajouter une image voici la syntaxe <img src="adresse_de_l_image" alt="texte altenatif" >La valueur de l'attribut alt sera affiché au lieu de l'image s'il y a une erreur lors du chargement. Elle peut également servir pour les aveugles.

On peut ajouter les attributs height et length pour modifier les dimensions. On verra dans un tutoriel plus avancé sur le CSS plus de détails sur les dimensions.
